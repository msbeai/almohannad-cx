# دليل تحرير almohannad.cx بنفسك

موقعك ملفات نصية بسيطة على جهازك في المجلد:
`~/Projects/almohannad-cx`

افتح أي ملف بمحرر نصوص (TextEdit يكفي، و[VS Code](https://code.visualstudio.com) أفضل)، عدّل، احفظ، ثم انشر. لا حاجة لمعرفة برمجية.

---

## القاعدة الذهبية: التعديل ثم النشر

أي تعديل تحفظه لا يظهر على الإنترنت إلا بعد النشر. النشر كله أمر واحد:

```bash
cd ~/Projects/almohannad-cx && ./publish.sh
```

افتح تطبيق **Terminal**، الصق السطر أعلاه، اضغط Enter، وانتظر رسالة "✅ تم النشر".

ولو أردت **معاينة قبل النشر**: شغّل `npm run dev` داخل مجلد المشروع وافتح `http://localhost:4321` في المتصفح — كل تعديل تحفظه يظهر فورًا في المعاينة. أوقف المعاينة بـ Ctrl+C.

## إضافة مقال جديد — خطوة بخطوة

1. أنشئ ملفًا جديدًا في `src/content/ar/` باسم إنجليزي قصير بشرطات، مثل: `my-new-article.md` (هذا الاسم يصبح رابط المقال).
2. الصق هذا القالب في أوله وعدّل القيم:

```markdown
---
title: "عنوان المقال بالعربية"
titleAr: "عنوان المقال بالعربية"
titleEn: "English Title"
theme: cx
order: 60
linkedin: ""
date: 2026-06-15
---

## تمهيد

نص المقال هنا. سطر فارغ بين كل فقرة وأخرى.

## عنوان فرعي

- نقطة أولى
- نقطة ثانية
```

ملاحظات:
- `theme:` إما `cx` (تجربة العميل) أو `research` (الأبحاث التسويقية).
- `order:` رقم تسلسلي — اجعله أكبر رقم موجود في نفس المحور + 1.
- الصور: ضع الصورة في `public/images/articles/` ثم أدرجها في النص هكذا: `![](/images/articles/اسم-الصورة.jpg)`
- **الغلاف يتولد تلقائيًا** عند النشر — لا تحتاج لفعل شيء.
- للترجمة الإنجليزية: أنشئ ملفًا **بنفس الاسم تمامًا** في `src/content/en/`.

## إضافة كتاب

افتح `src/data/books.json` وأضف داخل القائمة `"books": [ ]`:

```json
{
  "titleAr": "اسم الكتاب",
  "titleEn": "Book Title",
  "year": "2024",
  "descriptionAr": "وصف قصير للكتاب.",
  "descriptionEn": "Short description.",
  "link": "https://رابط-الكتاب",
  "cover": "/images/books/cover.jpg"
}
```

(ضع صورة الغلاف في `public/images/books/`. الفاصلة `,` بين كل كتاب وآخر.)

## إضافة حلقة بودكاست

افتح `src/data/podcasts.json` وأضف داخل `"episodes": [ ]`:

```json
{
  "titleAr": "عنوان الحلقة",
  "titleEn": "Episode Title",
  "show": "اسم البودكاست",
  "year": "2025",
  "youtube": "https://www.youtube.com/watch?v=XXXXXXXX",
  "descriptionAr": "وصف قصير."
}
```

الفيديو يظهر مضمّنًا في الصفحة تلقائيًا.

## تحذيرات مهمة

1. **لا ترفع المشروع إلى GitHub عام** — خطوط ثمانية داخل المشروع وترخيصها يمنع إعادة توزيعها. (مستودع خاص لا بأس به).
2. في ملفات `.md`: لا تحذف السطور بين `---` و `---` في أول الملف — هذه بيانات المقال.
3. في ملفات `.json`: انتبه للفواصل — فاصلة ناقصة أو زائدة تكسر الصفحة. (المعاينة ستكشف ذلك قبل النشر).
4. لو كسرت شيئًا ولم تعرف السبب: كل شيء محفوظ في git — اطلب مني "أرجع آخر نسخة سليمة" وسأتولى الأمر.

## للمستقبل (اختياري)

لو أردت تحرير المقالات من المتصفح مباشرة (بدون Terminal): يمكن ربط المشروع بمستودع GitHub **خاص** + النشر التلقائي من نتلفاي — أي تعديل تحفظه من موقع GitHub ينشر تلقائيًا خلال دقيقتين. قل لي متى أردت تفعيلها.
